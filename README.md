# Customer Analytics Dashboard
![Dashboard Screenshot](image-name.png)
## Overview
This project focuses on analyzing customer behavior and business performance using an interactive Power BI dashboard. The goal is to extract meaningful insights that help improve decision-making and customer retention.

## Key Features
- Customer segmentation (VIP, Premium, Regular)
- Revenue and sales trend analysis
- Customer churn rate tracking
- Customer lifetime value (CLV) analysis
- Monthly performance insights

## Tools & Technologies
- Power BI
- SQL
- Microsoft Excel

## Insights
- Identified high-value customer segments contributing the majority of revenue
- Analyzed customer retention patterns and churn trends
- Observed consistent monthly growth in revenue
- Evaluated purchase frequency and customer lifetime value

## Outcome
Developed an interactive dashboard that enables businesses to monitor key performance indicators (KPIs) and make data-driven decisions.
